# ArkAI — AI Architecture Intelligence

AI-powered architectural intelligence platform for design, cost, sustainability, and compliance.

## Project info

ArkAI is a comprehensive suite for architects to manage land intelligence, floor plan generation, structural analysis, and more.

## Development

The only requirement is having Node.js & npm installed.

Follow these steps:

```sh
# Step 1: Install the necessary dependencies.
npm i

# Step 2: Start the development server with auto-reloading and an instant preview.
npm run dev
```

## Technologies

This project is built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
- Supabase
- Three.js (@react-three/fiber)

## Deployment

Deploy using standard Vite/React deployment workflows (Vercel, Netlify, or self-hosted).
